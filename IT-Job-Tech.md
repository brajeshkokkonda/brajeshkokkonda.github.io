# IT Job Roadmap & Guidance

## Full Stack Java Developer
### Front End (UI)
   - React/Angular
### Back End (API)
   - Java 25
   - Spring Boot Microservices, Spring Data JPA (Hibernate), Spring Security (OAuth2, JWT), OpenAPI/Swagger, Spring Cloud, Spring AI
   - DB: SQL (MSSQL/PostgreSQL/MySQL), NOSQL (MongoDB/Cassandra/DynamoDB), Graph DB(Neo4J), Redis Cache, Cloud Cache
   - Event Driven Architecture: Apache Kafka, gRPC, MQ
   - Unit Testing: JUnit, Mockito, TestContainers (unit/functional/integration), TDD/BDD
   - Tools: Git, Maven, Postman, Bash, Python, IntelliJ IDE, Log monitoring (Splunk/BigPanda), Gitlab/Github/Bitbucket, Github Copilot 
### Cloud
   - Azure: functions
   - Google Cloud: Kubernetes
   - AWS: lambda, s3, ec2
### DevOps
   - CI/CD: GitLab, Jenkins, Azure DevOps
   - Docker, Kubernetes
   - Ansible/Terraform
### Other skills:
   - Datastructures & Algorithms, Problem Solving, Design & Architectural Patterns, SOLID, Clean Code Principles & Best Practices, software development principles, ALM tools, software methodologies like agile/scrum/kanban
   - Excellent communication and collaboration skills, with the ability to work effectively in a team environment or as an individual contributor when required.
